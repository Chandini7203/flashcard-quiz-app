# Flashcard Quiz App

A simple, stylish web application to create, view, and manage flashcards for learning and revision.  
Built with HTML, CSS, and JavaScript.

## Features

- Add new flashcards with questions and answers
- Flip cards to reveal answers
- Delete cards easily
- Modern, responsive design with attractive styles
- All code is beginner-friendly and well organized

## How to use

1. Open `index.html` in your browser.
2. Add a question and answer using the form.
3. Click "Flip" to view the answer or question.
4. Click "Delete" to remove unwanted flashcards.

## Technologies Used

- HTML
- CSS (includes gradient backgrounds, card styles, and responsive layout)
- JavaScript (used for dynamic flashcard management)
